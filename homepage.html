Welcome to a color coordinate generator website I created for a course!


To navigate the website, use the navigation bar on the left.

    • Use the "Color Selection" button to interact with the database. You can add a color to the database, delete a color, and edit an existing color. There are some restrictions that you might see when inteeracting with it, but that is for you to find out!
      Use the "Color Coordinate Generator" button to interact with the generator where you can populate the table cells with colors you select. The selection is done through the upper table and it will display the colors in the database. Before this screen, there is a form where you can select the size of the lower tables and the select how many colors you want for the upper table. On the second page, there is a button that shows you a printable page of the current page with all colors selected populated in the the upper table.
